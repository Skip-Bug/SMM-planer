# Публикация постов в Одноклассники (OK.ru)

Этот скрипт на Python позволяет публиковать текстовые записи и изображения в группы Одноклассников через официальное API. Также поддерживает удаление созданных публикаций.

## Возможности

- Публикация текстового поста в группе
- Публикация изображения с подписью
- Удаление поста по его идентификатору
- Автоматическая генерация подписи (`sig`) для авторизации запросов

## Требования

Установите зависимости:

```bash
pip install requests python-dotenv
