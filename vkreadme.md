# Публикация постов во Вконтакте

Этот скрипт позволяет публиковать сообщения в группе вместе с картинкой. Взаимодействует с оффициальным апи ВК.ru. Присутствует возможность удаления

 ## Возможности

- Публикация текстового поста в группе
- Публикация изображения
- Удаление поста по его идентификатору

## Требования

Установите зависимости:

```bash
pip install -r requirements.txt```
